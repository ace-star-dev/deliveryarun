# ARUNA Asian Bar - Delivery Webapp

Una experiencia sensorial que trasciende lo ordinario. Este es o sistema de delivery premium para **Aruna Asian Bar**, localizado em Salto del Guairá, Paraguay.

## 🍱 Visión General

El webapp combina la sofisticación de la culinaria asiática de alto patrón con una experiencia digital fluida y moderna.

### Características Principales:
*   **Aesthetic Luxury:** Paleta de colores Black Obsidian, Burgundy Red y Old Gold.
*   **Menú de Autor:** Selección premium de Sashimi, Wok, Ramen y Coctelería Signature.
*   **Dual Checkout:** Sistema integrado para pedidos directos vía WhatsApp o formulario interno de delivery.
*   **Totalmente Responsivo:** Experiencia optimizada para dispositivos móviles.

## 🛠️ Tecnologías

*   **React 19**
*   **Vite 8**
*   **Lucide React** (Iconografía)
*   **Agent Skills** (Vercel Labs)

## 🧠 Agent Skills

Este proyecto utiliza el ecosistema de **Vercel Labs Skills** para asegurar los más altos estándares de desarrollo y diseño.

Skills instalados localmente:
1.  **`vercel-react-best-practices`**: Directrices de ingeniería de Vercel para React.
2.  **`web-design-guidelines`**: Estándares de diseño web premium.

Si estás trabajando en este proyecto con un agente de IA (como Gemini CLI, Claude Code o Cursor), el agente utilizará automáticamente estas instrucciones para mantener la calidad del código y la estética.

## 🚀 Cómo Empezar

### Instalación
```bash
npm install
```

### Desarrollo
```bash
npm run dev
```

### Build para Producción
```bash
npm run build
```
